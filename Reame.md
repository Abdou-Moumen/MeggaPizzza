## 📋 Overview

MeggaPizza is a modern, animated landing page designed for a pizza delivery brand. It showcases menu items, special offers, and brand personality with smooth scroll-triggered animations, interactive elements, and a mobile-first responsive design.

## 🎯 Key Features

### 🍕 Visual Highlights
- Hero section with animated pizza slice floating effect
- Animated menu cards with hover effects
- Count-up statistics (happy customers, pizza varieties, delivery zones)
- Parallax scrolling effects
- Smooth fade-in animations on scroll

### 📱 User Experience
- Fully responsive (mobile, tablet, desktop)
- Smooth navigation with active link highlighting
- Click-to-call phone button
- Google Maps integration (location embed)
- Newsletter subscription form

### 🎨 Interactive Elements
- GSAP-powered text reveals
- Staggered menu item animations
- Hover-triggered pizza toppings animation
- Animated CTA buttons with pulse effects
- Scroll-triggered counter animation
